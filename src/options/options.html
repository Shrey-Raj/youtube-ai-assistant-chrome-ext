Get Your Gemini API Key

Follow these steps in Google Cloud Console and AI Studio:

  • Go to Google Cloud Console.
  • Click the project dropdown in the header and select New Project.
  • Enter a project name (e.g. “YouTube Summarizer”) and click Create.
  • Once your project is ready, navigate to Google AI Studio API Keys.
  • Sign in with the same Google account you used in Cloud Console.
  • Click Create API Key. A popup will appear.
  • In the popup, select the project you just created.
  • Click Create API Key in that project.
  • Copy the generated key and paste it into the settings form below.
Your key is stored locally in your browser and never shared. Manage your keys
